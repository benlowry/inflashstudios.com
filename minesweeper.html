I made this game for three reasons. I had a decade of experience using Flash but hasn't migrated to the newest version of Actionscript, I wanted to make games using Flash (in case you haven't guessed), and Minesweeper doesn't really have any hard bits. It was a good introduction to AS3 so coming on a year later I've produced a tutorial to accompany it so you can learn too.
